# 📅 Calendar Agent 설치 가이드

문자·카카오톡·Gmail로 오는 메시지에서 **일정을 자동으로 찾아 캘린더에 등록**해 주는 앱이에요.
모든 분석은 **폰 안에서만** 이뤄지고, 메시지 내용은 어디로도 전송되지 않습니다. (인터넷 없이도 동작)

---

## 0. 준비물

받으셨을 파일 **2개**:
- **`app-debug.apk`** — 앱 설치 파일
- **`calendar.Q4_K_M.gguf`** — 일정 인식 AI 모델 (약 380MB)

> 폰은 **안드로이드 13 이상**이어야 합니다. (아이폰은 안 됩니다.)
> 두 파일은 카톡으로 안 보내질 수 있어요(용량). 구글 드라이브 링크나 USB로 받는 걸 권장합니다.

---

## 1. 앱 설치

1. 받은 **`app-debug.apk`** 파일을 탭합니다.
2. "이 출처의 앱 설치" 를 허용해 달라는 안내가 나오면 → **설정으로 이동 → 이 출처 허용** 후 뒤로.
3. **설치** 를 누릅니다.
4. "Play 프로텍트가 인식하지 못한 앱" 경고가 나오면 → **자세히 → 무시하고 설치** (개인이 만든 앱이라 그렇습니다).
5. 설치되면 **Calendar Agent** 아이콘이 생깁니다.

---

## 2. 처음 설정 (앱을 열면 자동 안내)

앱을 처음 열면 **"처음 설정"** 체크리스트가 뜹니다. 위에서부터 차례로 ✓ 표시가 될 때까지 눌러 주세요.

1. **모델 (.gguf) — 임포트**
   → 받은 **`calendar.Q4_K_M.gguf`** 파일을 선택합니다. (잠시 복사 후 ✓)
2. **알림 — 허용** (알림 표시 권한)
3. **캘린더 — 허용** (일정을 캘린더에 넣기 위해)
4. **문자(SMS) — 허용** (문자에서 일정 찾기)
5. **카카오톡·Gmail 알림 접근 — 허용**
   → "알림 접근" 설정 화면이 열리면 목록에서 **Calendar Agent** 를 찾아 **켜기**.
   (카카오톡과 Gmail은 이 권한 하나로 둘 다 수집됩니다.)

모두 ✓ 가 되면 **시작하기** 를 누릅니다.

> 💡 **Gmail**은 별도 로그인이 필요 없습니다. Gmail 앱의 **알림이 켜져 있기만** 하면 됩니다.
> (설정 → 앱 → Gmail → 알림 이 켜져 있는지 확인)

---

## 3. (권장) 백그라운드에서 안 끊기게

폰을 안 보고 있어도 메시지가 오는 즉시 처리되게 하려면:

- 앱 → **설정** →
  - **자동 수집 상주** 켜기 (수집 중 알림이 항상 표시됩니다)
  - **배터리 최적화 제외 요청** → 허용

---

## 4. 사용법

메인 화면 **"오늘 받은 일정"**:

- 위쪽 **카톡 / 문자 / Gmail / 예비** 숫자 = 오늘 현황
- **예비**(노란 배지): 앱이 찾았지만 확인이 필요한 일정 → **[등록]** 누르면 캘린더에 추가, **[편집]** 으로 고치고, **[무시]** 로 버림
- **등록**(보라 배지): 오늘 캘린더에 넣은 일정 → **[취소]** 로 되돌리기
- **신뢰도가 높은 일정은 자동으로 등록**되고, 알림으로 "되돌리기"를 줍니다. (자동 등록 기준은 설정에서 조절 가능)

일정이 잘못 잡히면 **[편집]** 에서 고쳐서 등록하면 됩니다.

---

## 5. 잘 안 될 때

| 증상 | 확인 |
|---|---|
| 일정이 아예 안 잡혀요 | 처음 설정의 5가지 권한이 모두 ✓ 인지, 모델이 임포트됐는지 |
| 카톡/Gmail이 안 잡혀요 | **알림 접근**이 켜져 있는지 + 해당 앱(카톡·Gmail)의 **알림이 켜져** 있는지 |
| 폰 잠그면 안 돼요 | 설정에서 **자동 수집 상주** 켜기 + **배터리 최적화 제외** |
| 긴 메일의 일정을 놓쳐요 | Gmail은 **제목과 미리보기**만 봅니다. 일정이 본문 깊이 있으면 못 볼 수 있어요(정상) |

---

## 개인정보

- 메시지·메일 내용은 **폰 안에서만** 분석되고 외부로 전송되지 않습니다.
- 캘린더 등록은 폰의 기본 캘린더(구글 캘린더 등)에 직접 들어갑니다.
- 자동 수집을 끄려면 설정에서 채널 토글을 끄거나, 앱을 삭제하면 됩니다.

궁금한 점은 앱을 보내준 분께 문의하세요. 🙂
